🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Problem: "Invalid email or password"
Solution: 
  - Make sure you're using the temporary password exactly as shown
  - Password is case-sensitive
  - Try registering a new test member

Problem: Temporary password not showing after registration
Solution:
  - Check browser console for errors
  - Verify join-us.php was uploaded correctly
  - Clear browser cache

Problem: Can't access Member Portal
Solution:
  - Make sure you're logged in first
  - Check session cookies are enabled
  - Try logging in again at /member-login

Problem: Old site still showing
Solution:
  - Hard refresh browser (Ctrl+Shift+R / Cmd+Shift+R)
  - Delete old CSS/JS files from server
  - Clear browser cache completely
