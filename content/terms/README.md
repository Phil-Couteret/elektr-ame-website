# Terms and Conditions Content

Edit these files to update the terms and conditions displayed at `/terms-and-conditions`:

| File | Language |
|------|----------|
| `terms-en.ts` | English |
| `terms-es.ts` | Spanish |
| `terms-ca.ts` | Catalan |

Each file exports the full terms content. The site uses the file matching the user's selected language.

**To edit:** Open the relevant file, change the text strings, save, and redeploy.
